# Undergrad_Research_Mission
학부연구생 과제 저장소입니다.

## 과제 정의
* 주제: 자율주행 데이터(LiDAR 포인트 클라우드와 카메라 이미지) 정렬
* 목적: 
  - 서로 다른 위치에 설치된 LiDAR 센서와 카메라 센서 간의 좌표계 변환 과정을 이해합니다.
  - 제공된 여러 센서 변환 행렬(C2에서 C0, C0에서 LiDAR)을 수학적으로 조합하여, LiDAR의 3D 데이터 점들을 카메라 2번(C2) 화면 위 정확한 픽셀 위치에 투영(Projection)시키는 파이썬 코드를 완성하는 것이 목표입니다.

## 진행 상황 및 학습 기록

### 1. 가상환경 구성 및 개발 환경 세팅 (2026-06-25 완료)
과제 수행 중 기존 파이썬 환경과의 의존성 충돌을 방지하기 위해 가상환경을 생성하고, 데이터 처리에 필요한 핵심 라이브러리 설치를 완료했습니다.
* **가상환경 이름:** `.venv`
* **설치한 주요 패키지:**
  - `numpy`: 3D 포인트 클라우드 좌표 계산 및 외부/내부 파라미터 행렬 연산용
  - `opencv-python` (`cv2`): 원본 이미지 로드 및 이미지 크기 분석용
  - `matplotlib`: 최종 이미지 위에 LiDAR 점들을 rainbow 컬러맵으로 투영하여 시각화하는 용도

### 2. 데이터셋 및 캘리브레이션 파일 배치 완료 (2026-06-25 완료)
코드(`problem_1.py`)가 실행될 때 원본 데이터와 정렬 파라미터를 올바르게 참조할 수 있도록 프로젝트 파일 구조를 다음과 같이 구성했습니다.
* 센서 데이터 구조:
  - `0000000000.png`: 카메라 2번으로 촬영된 전방 원본 이미지 데이터
  - `0000000000.bin`: LiDAR로 측정된 전방 360도 3D 포인트 클라우드 바이너리 데이터
